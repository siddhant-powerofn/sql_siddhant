1) Write a SQL query to find student who got placed with highest package, along with the name of the company.	 

answers: SELECT STUDNET.NAME, COMPANY.NAME, PLACEMENTS.PACKAGE FROM STUDENT JOIN COMPANY JOIN PLACEMENT ON STUDENT.ID = PLACEMENTS.S_ID AND COMPANY.ID = PLACMENTS.C_ID WHERE PACKAGE = (SELECT MAX(PACKAGE) FROM PLACEMENTS);

2) HOD of ECE department wants to know placement details of all his department students, Write a SQL query for this scenario

answers: SELECT STUDENT.NAME, STUDENT.DEPARTMENT, IF(STUDENT.ID IN (SELECT S_ID FROM PLACEMENTS JOIN STUDENT ON PLACEMENTS.S_ID = STUDENT.ID), "YES", "NO") AS PLACED, COMPANY.NAME AS COMPANY, PLACEMENTS.PACKAGE FROM STUD
ENT LEFT JOIN PLACEMENTS ON STUDENT.ID = PLACEMENTS.S_ID LEFT JOIN COMPANY ON PLACEMENTS.C_ID = COMPANY.ID WHERE STUDENT.DEPARTMENT='ECE' ORDER BY STUDENT.NAME;

3) Write a SQL query to display total number of students each company has hired.

ansets: SELECT COMPANY.NAME, COUNT(COMPANY.NAME) FROM COMPANY INNER JOIN PLACEMENTS ON COMPANY.ID = PLACEMENTS.S_ID GROUP BY COMPANY.NAME;

4) Write a SQL query which displays list of companies conducted interview in month of september.

answers: SELECT MONTHNAME(INTERVIEW_DATE) AS MONTH, NAME AS COMPANY FROM COMPANY WHERE MONTHNAME(INTERVIEW_DATE)='SEPTEMBER';

5) Write a SQL query which displays list of companies which conducted interview till date.

7)Accenture would like to shortlist candidates with CGPA > 7 and no need of student who already got placed, write SQL query that statisfies their given criteria

answers: SELECT NAME AS STUDENT, DEPARTMENT, CGPA FROM STUDENT WHERE STUDENT.ID NOT IN (SELECT PLACEMENTS.S_ID FROM PLACEMENTS) AND STUDENT.CGPA > 7;
